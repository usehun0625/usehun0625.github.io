학습 차원에서 다음과 같은 동작을 하는 자동화를 구현하고 구현 과정을 문서로 정리한다.

  • 매시간마다 6시간 전까지의 이벤트 데이터 개수를 시간 단위로 집계한다.
  • 집계된 내용은 Slack Bot으로 매시간마다 보고 한다.

아래의 구현 과정은 Elasticsearch에 event_data_index 인덱스에 이벤트 데이터가 저장된 것을 가정하고 진행한다.

테스트 환경

  • OS : macOS Catalina v10.15.2
  • Python : v3.6.5
  • Chalice : chalice 1.9.1
