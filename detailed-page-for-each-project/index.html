Histogram

  • Image Buffer의 값 분포를 알려주는 메서드
  • 8, 16 (Bit) Depth와 1, 3, 4 Channel을 지원
  • 일반적인 방법보다 1.53배 빠른 퍼포먼스 갖는 알고리즘 사용
  • Link
    • Histogram Fast 알고리즘
  Histogram을 그래프로 도식화한 이미지

FFT

  • 이미지의 Fast Fourier Transform한 값을 구해주는 메서드
  • 이미지의 FFT한 값에 Butterworth filter 등을 사용하면 Image Recognition을 용이하게 할 수 있음
  • Float type, Double type을 지원
  • DIF 연산법을 사용하여 일반적인 방법보다 1.39배 빠른 퍼포먼스
